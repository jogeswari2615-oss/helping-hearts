 Helping Hearts ❤️

Helping Hearts is a full-stack, location-aware web application built with Django and MySQL to streamline charitable giving across Tamil Nadu. The platform bridges the gap between donors and verified orphanages by replacing blind donations with an interactive, demand-driven workflow. Using an integrated map facility, users can locate nearby orphanages, select a specific institution, and view a dynamic wishlist of exact products or financial aid needed at that moment. To maximize donor convenience, the application captures necessary logistics details and offers two flexible fulfillment options: self-delivery to the facility or a scheduled pickup organized by the orphanage.


📌 Problem and Solution

The Problem: Donors guess what orphanages need, causing surpluses of some goods and shortages of essentials, while lack of transport discourages giving.

The Solution: Helping Hearts uses a map to locate Tamil Nadu orphanages and display their real-time needs. It removes logistics hurdles by offering choice of self-delivery or orphanage pickup.


 🛠️ Tech Stack

Backend: Django (Python)
Database: MySQL
Frontend: HTML5, CSS3, JavaScript
